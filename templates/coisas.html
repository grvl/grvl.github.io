Legenda
Ônibus
As linhas de ônibus são representadas pelas linhas coloridas no mapa. A cor da linha representa a nota da linha, e a nota é dada proporcionalmente ao número de ônibus acessíveis na linha. Clique na linha para ver seu nome e nota.
0% vermelho, 50% amarelo, 100% verde
 Zona Azul
Cada pin representa vagas de zona azul reservadas para idosos ou pessoas com deficiência. Clique no pin para ver qual seu tipo e endereço.
 Selo de acessibilidade
Cada pin representa um estabelecimento com o selo de acessibilidade da prefeitura. Clique no pin para ver seu nome.
Guiaderodas
Cada pin com uma estrela representa um estabelecimento com nota no Guiaderodas. A cor do pin representa a nota do estabelecimento, sendo vermelho estabelecimentos com menos de 1.7, laranja estabelecimentos entre 1.7 e 3.6, e verde estabelecimentos com 3.6 ou mais. Clique no pin para ver seu nome e sua nota.
 Metrô
Cada pin representa uma estação de metrô acessível. Clique no pin para ver seu nome.
 CPTM
Cada pin representa uma estação de CPTM acessível. Clique no pin para ver seu nome.
